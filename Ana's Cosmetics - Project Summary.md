## 🔧 Customization Guide

### Change Brand Colors
Edit CSS variables in `assets/css/styles.css`:
```css
--accent-gold: #d4af37;
--secondary-purple: #8b5a8e;
--accent-rose: #e8b4c8;
```

### Update Contact Information

### Add/Edit Products
Edit `assets/js/products.js` and add product objects:
```javascript
{
  id: 61,
  name: 'Product Name',
  category: 'Category',
  price: 49.99,
  // ... other properties
}
```

### Change WhatsApp Number
Edit `assets/js/utils.js` function `getWhatsAppLink()`:
```javascript
return `https://wa.me/YOUR_NUMBER?text=...`;
```
