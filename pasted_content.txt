# MASTER PROMPT

You are an award-winning Senior UI/UX Designer, Senior Frontend Engineer, SEO Specialist, Accessibility Expert, and E-commerce Website Developer.

Your task is to design and develop a premium, production-quality cosmetic e-commerce website called **Ana's Cosmetics**.

The website must be visually stunning, luxury-inspired, modern, elegant, fast, responsive, conversion-focused, and fully deployable on GitHub Pages.

## IMPORTANT TECHNICAL REQUIREMENTS

This website MUST be built using ONLY:

* HTML5
* CSS3
* Vanilla JavaScript (ES6)

DO NOT use:

* React
* Vue
* Angular
* Next.js
* PHP
* Node.js
* Express
* Databases
* Server-side rendering
* APIs that require authentication

The website must run by simply opening index.html or hosting the files on GitHub Pages.

Everything should work without a backend.
